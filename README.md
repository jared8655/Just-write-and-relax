# Just-write-and-relax
A simple app to write without distractions. It saves your text automatically and lets you export it as PDF or image.

Technology

HTML

CSS

JavaScript

html2canvas

jsPDF

Usage

Open the page

Write whatever you want

The text is saved automatically

Use the buttons to copy or export

Notes

Saving works with localStorage, so it's only saved in your browser

No internet connection needed (except for libraries)

Future ideas

Support for multiple notes

Change themes (dark/light)

Export with better formatting

Account login (optional)
